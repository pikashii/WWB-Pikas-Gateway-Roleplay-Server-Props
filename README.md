# [WWB] Pika's Gateway Roleplay Server Props

All of our assets from What We Become Part II: What Remains, archived in one convenient mod.

This is a companion module to [Pika's Gateway Roleplay Server Props](https://github.com/pikashii/Pikas-Gateway-Roleplay-Server-Props) — everything specific to the What We Become II: What Remains lore has been split out from the main mod into this standalone package.

## Features

- **NPC Props** — Placeable character sprites for roleplay NPCs, with corpse variants and multi-pose support
- **Monster Items** — Collectible visual sprites for various creatures
- **Vehicle Props** — Decorative items representing vehicles
- **Fast Travel Points** — Recipe-based teleportation system for navigating between in-world locations (Canada, Celephais, Pittsford, Texas, and more)
- **Admin Fast Travel** — Privileged teleport key granting access to all locations
- **Examine UI Integration** — Character portraits and descriptions viewable in-game via the Examine system

## Requirements

This mod requires the following:

- [Pika's Gateway Roleplay Server Props](https://steamcommunity.com/workshop/filedetails/?id=2708362223) (and all of its dependencies)
  - Spongie's Clothing
  - [REUPLOAD] SMUI, SLEO, & SRPS
  - KATTAJ1 Clothes Core [B41/B42MP]
  - KATTAJ1 Military Pack [B41/B42MP]
  - Filibuster Rhymes' Used Cars! - Animation Beta

## Installation

### Steam Workshop

Subscribe to the mod on the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=2949227010).

### Manual

Clone this repository into your Project Zomboid mods directory:

```
%USERPROFILE%\Zomboid\mods\
```
