# [WWB] Pika's Gateway Roleplay Server Props

All of our assets from What We Become Part II: What Remains, organized into optional modules.

This is a companion to [Pika's Gateway Roleplay Server Props](https://github.com/pikashii/Pikas-Gateway-Roleplay-Server-Props) — everything specific to the What We Become II: What Remains lore has been split out from the main mod into this standalone package.

## Modules

| Module | Mod ID | Required? | Description |
|--------|--------|-----------|-------------|
| **! WWB Core** | `PikasGatewayPropsWWB` | Yes | Vehicle props, poster, and shared foundation. Requires the B41 core mod. |
| **NPC Module** | `PikasGatewayPropsWWB-NPC` | No | 70+ placeable NPC props with corpse variants, multi-pose support, and Examine UI portraits. |
| **Fast Travel Module** | `PikasGatewayPropsWWB-FTP` | No | Recipe-based teleportation system across Canada, Celephais, Pittsford, Texas, and more. Includes admin fast travel. |
| **Monster Module** | `PikasGatewayPropsWWB-Monster` | No | 32+ collectible monster sprites. |

All optional modules require the WWB Core module.

## Requirements

- [Pika's Gateway Roleplay Server Props](https://steamcommunity.com/workshop/filedetails/?id=2708362223) (and all of its dependencies)
  - Spongie's Clothing
  - [REUPLOAD] SMUI, SLEO, & SRPS
  - KATTAJ1 Clothes Core [B41/B42MP]
  - KATTAJ1 Military Pack [B41/B42MP]
  - Filibuster Rhymes' Used Cars! - Animation Beta

## Installation

### Steam Workshop

Subscribe to the mod on the [Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=2949227010).

### Manual

Clone this repository into your Project Zomboid mods directory:

```
%USERPROFILE%\Zomboid\mods\
```

Each subdirectory is a separate mod — enable the ones you need in the Project Zomboid mod loader.
